# Коды шахмат
Каждая фигура храниться в 4 битах.

Белые фигуры:
- 0001 - Король (King) - k
- 0010 - Королева (Queen) - q
- 0011 - Слон (Bishop) - b
- 0100 - Конь (Knight) - n
- 0101 - Ладья (Rook) - r
- 0110 - Пешка (Pawn) - p


Чёрные будут инвертированными белыми.

Первый бит означает цвет.
- 0000 - пустая клетка.
